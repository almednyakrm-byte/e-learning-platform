# منصة تعليم إلكترونية شاملة مع دورات وامتحانات وتقييم

## ملخص المشروع

منصة تعليم إلكترونية شاملة مع دورات وامتحانات وتقييم، تهدف إلى توفير بيئة تعليمية متكاملة لطلاب الجامعات والكليات، حيث يمكنهم الحصول على الدورات التعليمية والامتحانات والتحليلات الشخصية.

## هيكل المشروع

### مكونات المشروع

* `backend`: مكون الخلفية الذي يحتوي على كود البرنامج والبيانات.
* `frontend`: مكون الجانب الأمامي الذي يحتوي على الواجهة المستخدمة.
* `database`: قاعدة البيانات التي تحتوي على البيانات الخاصة بالطلاب والكورسات.

## إعداد البيئة باستخدام docker-compose

### خطوات الإعداد

1. تأكد من أنك تمتلك Docker وdocker-compose مثبتين على جهازك.
2. افتح المجلد الرئيسي للمشروع في командية.
3. استخدم الأمر التالي لتشغيل البيئة باستخدام docker-compose:

bash
docker-compose up


### تفعيل الخدمات

* افتح متصفح الإنترنت واكتب `http://localhost:5000` لفتح الواجهة المستخدمة.
* افتح متصفح الإنترنت واكتب `http://localhost:5001` لفتح لوحة التحكم.

## مكونات البرنامج

### الدورات

* `Course`: تضمين الدورات التعليمية.
* `Lesson`: تضمين الفصول الدراسية.
* `Assignment`: تضمين المهام والامتحانات.

### الطلاب

* `Student`: تضمين بيانات الطلاب.
* `Enrollment`: تضمين التسجيل في الدورات.

### الأدوار

* `admin`: دور الإدارة.
* `teacher`: دور المعلم.
* `student`: دور الطالب.
